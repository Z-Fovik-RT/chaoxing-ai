# 学习通 · AI智脑Pro

> 原名「学习通AI助手」，已更名为「学习通 · AI智脑Pro」
> 作者：Z-Fovik-RT | [GitHub](https://github.com/Z-Fovik-RT/chaoxing-ai) | [脚本猫](https://scriptcat.org/zh-CN/script-show-page/6369)

学习通全自动学习助手 — 10大AI模型 + 双题库 + 视频自动播放 + 字体解密 + 章节导航 + 题目一键复制

***

## 功能

- **智能答题**：10大AI模型（DeepSeek/通义千问/智谱GLM/Kimi/OpenAI/Claude等）、40+模型可选、五层答案匹配、字体解密、垃圾答案拦截
- **双题库**：lyck6.cn + icodef.com 双源覆盖，题库优先 + AI兜底，单题库无答案自动切换下一源
- **视频/音频**：自动播放、倍速（1x~2x）、复习模式补挂时长、视频内题目自动处理
- **测验考试**：自动答题、智能提交（全答完才交卷/有未答自动保存并跳下一张）、强制提交（无论是否答完都提交）、考试跳题、好学生模式、重做模式
- **章节导航**：自动识别未完成任务、智能跳转下一节、闯关模式兼容
- **页面增强**：粘贴解锁、一键复制题目、浮窗设置面板、日志面板
- **防检测**：防休眠（Web Audio API）、多设备监控绕过
- **字体解密**：opentype.js + TyprMd5 + 多源字体映射表

## 配置

在脚本浮窗中可实时调整所有配置，常用项：

| 配置项 | 默认 | 说明 |
|--------|------|------|
| `video` / `audio` | `1` | 自动处理视频/音频 |
| `rate` | `1` | 播放倍速（1/1.25/1.5/2） |
| `work` | `1` | 自动处理测验 |
| `decrypt` | `1` | 字体解密（推荐开启） |
| `sub` | `0` | 自动提交（全部答完才提交，有未答则自动保存并跳到下一张） |
| `force` | `0` | 强制提交（无论是否答完都提交，适合快速刷课时） |
| `searchEnabled` | `1` | 搜题总开关（实时生效） |
| `thirdPartyApi` | `0` | 启用第三方题库（lyck6.cn + icodef.com） |
| `reqIntervalTime` | `3` | 搜题间隔（秒），0为不节流，同时控制题目处理间隔 |

更多配置请查看脚本内 `cxaiCfg` 注释。
